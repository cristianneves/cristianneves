# 💫 About Me:
Com mais de 3 anos de experiência no ecossistema Java, meu foco é construir sistemas de alta disponibilidade e performance. Tenho uma sólida vivência no desenvolvimento de microsserviços utilizando Spring Boot e Quarkus, sempre prezando por arquiteturas limpas e código escalável.<br><br>Ao longo da minha trajetória, trabalhei entre o desenvolvimento web e a cultura DevOps, automatizando pipelines, testes e otimizando o ciclo de entrega de software. No lado dos bancos de dados, possuo experiência na modelagem, manipulação, e versionamento de bancos de dados em ambientes de produção, SQL e NoSQL.<br><br>E além de um apaixonado por tecnologia desde criança, um sou também um grande defensor da troca de conhecimento, que acredita que o melhor código é aquele construído através da colaboração e que facilita a vida de quem vai mantê-lo no futuro.
